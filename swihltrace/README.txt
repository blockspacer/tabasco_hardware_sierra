	This tool is used to capture the TRACE LOG
	==========================================

1. How to customize
===================
   
1.1 TRACE_PORT_NAM
    it is the port name which trace log information
    wil be outputed. You can redefine it according to your
    configuration at your modem side. 

1.2 TRACE_FILE
    it is a file which trace log information will be stored.
    You can redefine it, but make sure the trace log program
    has the write permission at the related directory.
