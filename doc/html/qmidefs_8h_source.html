33 /* Timeout guard interval. This value is the basic timeout applied by the API
34  * side whenever it calls icapi_rcv(). It is added to the application's
35  * specified timeout on API calls, while the caller's timeout is used directly

38  * icapi_rcv() times out, then this is some kind of system failure like the SDK
39  * process has died.
