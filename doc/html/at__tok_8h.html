int at_tok_index_int ( char **  p_cur,
int *  p_out,
int  index 
)

Parses the base 10 integer at 'index' in the AT response line and places it in *p_out.

  • all intermediate values are discarded.
  • 'index' is relative to the current position in the AT response line, so that index=0 is always the next value.

Returns 0 on success and -1 on fail. Updates *p_cur.

SWI_TBD: this function could be made generic, by taking a function pointer instead of explicitly calling at_tok_nextint() but that would make calling it rather cumbersone. If this function is needed for other values, then revisit the implementation.
