Detailed Description

Each type related to a response message has an associated callback function defined for it. This structure defines the contents of an entry in the array of callback functions.

Parameters
gpsCbkfp Pointer to the callback function associated with the 'type' value
gpsCbkfnNamep Pointer to a string identifying the type for debug logging purposes
