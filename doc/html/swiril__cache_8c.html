Author

Instructions For Adding New Cached RIL Command Item:

1) add to swiril_cache_table[]

2) add to swiril_cache_set()

3) add to swiril_main.c/processRequest()

4) call swiril_cache_RIL_onRequestComplete() or swiril_cache_set(), as appropriate, in the handler function for the RIL command

Return command response, and also cache the response.

This function is similar to the standard RIL_onRequestComplete(), and is intended as an easy to use substitute that also handles caching the response. This function should only be called for RIL_E_SUCCESS, as there is no result to cache otherwise.
