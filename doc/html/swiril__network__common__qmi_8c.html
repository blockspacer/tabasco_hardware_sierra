static RIL_RAT getRadioTechFromDataBearerTech ( ULONG  dataBearerTech)
static

The following data bearer related functions are for the MC8355 in UMTS mode, to identify network technologies above UMTS.

This is needed because on some networks, the data capability is not above UMTS, so this alternative method is needed. Map from select data bearer values to RIL_RAT values.
