unsigned long  length

String Copy (with max.) Copies not more than "length" characters (characters that follow a null character are not copied) from the array pointed to by srcp into the array pointed to by destp.

If copying takes place between objects that overlap, the behaviour is undefined.

Parameters
[out]*destppointer to location of memory to where string will be copied
[in]*srcppointer to location of memory from where string will be copied
lengthmaximum characters to copy
Returns
The value of destp
Abort:
none
Note
No pointer verification If the string pointed to by srcp is longer than length characters, the resultant array pointed to by destp will not be null terminated. If the string pointed to by srcp is shorter than length characters, null characters are continually copied to the string pointed by destp until "length" characters have been copied.
